The Assignment¶

Take a ZIP file) of images and process them, using a library built into python that you need to learn how to use. A ZIP file takes several different files and compresses them, thus saving space, into one single file. The files in the ZIP file we provide are newspaper images (like you saw in week 3). Your task is to write python code which allows one to search through the images looking for the occurrences of keywords and faces. E.g. if you search for "pizza" it will return a contact sheet of all of the faces which were located on the newspaper page which mentions "pizza". This will test your ability to learn a new (library), your ability to use OpenCV to detect faces, your ability to use tesseract to do optical character recognition, and your ability to use PIL to composite images together into contact sheets.

Each page of the newspapers is saved as a single PNG image in a file called images.zip. These newspapers are in english, and contain a variety of stories, advertisements and images. Note: This file is fairly large (~200 MB) and may take some time to work with, I would encourage you to use small_img.zip for testing.

import zipfile

from PIL import Image
import pytesseract
import cv2 as cv
import numpy

# loading the face detection classifier
face_cascade = cv.CascadeClassifier('readonly/haarcascade_frontalface_default.xml')

# the rest is up to you!
CONTACT_HEIGHT = 100
CONTACT_WIDTH = 100
CONTACTS_PER_ROW = 5


def image_contains_search_string(img):
    grey = img.convert('L')
    text = pytesseract.image_to_string(grey)
    # Look for exact matches, case sensitive, to match the assignment results
    return search_string in text


def faces_sheet(img):
    cvImg = numpy.array(img)
    # Detect the faces.
    # scaleFactor 1.3 on the original image (no conversion) seems to give the best results with 
    # a few false positives.
    faces = face_cascade.detectMultiScale(cvImg, scaleFactor=1.3)
    if len(faces) == 0:
        return None
    
    # Prepare the contact sheet
    rows = (len(faces) - 1) // CONTACTS_PER_ROW + 1
    size = (CONTACT_WIDTH * CONTACTS_PER_ROW, rows * CONTACT_HEIGHT)
    sheet = Image.new('RGB', size, 'black')
    
    # Put the faces on the contact sheet
    x, y = 0, 0
    for face in faces:
        faceImg = img.crop((face[0], face[1], face[0]+face[2], face[1]+face[3]))
        # Resize the face only if it is too big
        if faceImg.width > CONTACT_WIDTH or faceImg.height > CONTACT_HEIGHT:
            faceImg = faceImg.resize((CONTACT_WIDTH, CONTACT_HEIGHT))
        sheet.paste(faceImg, (x * CONTACT_WIDTH, y * CONTACT_HEIGHT))
        x += 1
        if x == CONTACTS_PER_ROW:
            x = 0
            y += 1
            
    return sheet


search_string = input("Enter the word to search for: ")

with zipfile.ZipFile("readonly/images.zip") as zf:
    nl = zf.namelist()
    for name in nl:
        with zf.open(name) as file:
            img = Image.open(file)
            if image_contains_search_string(img):
                print("Results found in file {}".format(name))
                sheet = faces_sheet(img)
                if sheet == None:
                    print("But there were no faces in that file!")
                else:
                    display(sheet)
